To install the plugin, do the following:
-Download the DLL from the Release section
-Open Dalamud Settings in Game
-Navigate to the Experimental tab
-In the Dev Plugin Locations section, click Select Dev Plugin DLL
-Select the DLL that was downloaded
-Click Save changes at the bottom right
